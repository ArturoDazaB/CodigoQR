# MTTO App
## Aplicacion para gestion de activos por codigo QR

La aplicación móvil “MttoApp” desarrollada por el departamento de digitalización industrial se encuentra orientada inicialmente a la creación, registro y lectura de códigos QR que serán asignados a los tableros de control y de servicios, permitiendo acceder a la información necesaria del tablero (activo) de manera remota.

Para el desarrollo de esta aplicación se seleccionó Xamarin Forms como framework / marco de trabajo (framework de código abierto, propiedad de Microsoft, utilizado para el desarrollo de aplicaciones móviles multi-plataforma como iOS, Android y Windows Phone), permitiendo que el desarrollo de un aplicativo para varias plataformas sea construido desde un único proyecto con un código fuente base. Es decir, los aplicativos de las plataformas seleccionadas compartirán el 90% del código fuente.
